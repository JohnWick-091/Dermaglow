# DermaGlow — Dermatology Clinic Website

A modern, multi-page demo website built for dermatology clinics. Designed to showcase what a real clinic site could look like — used as part of cold outreach to potential clients.

**Live site → [dermaglow-clinic.netlify.app](https://dermaglow-clinic.netlify.app)**

---

## Pages

| File | Description |
|------|-------------|
| `Index.html` | Homepage — hero, intro, CTA |
| `Treatments.html` | Treatment offerings |
| `Testomonial.html` | Patient testimonials |
| `Stats.html` | Clinic stats & trust signals |
| `FAQ.html` | Frequently asked questions |
| `Form.html` | Booking / contact form |

---

## Features

- Smooth scroll throughout
- Active booking system
- Navbar with section-based navigation
- Dark mode design
- Fully responsive (mobile-first)
- SEO meta tags

---

## Tech Stack

- Vanilla HTML & CSS
- Tailwind CSS (CDN)
- Google Fonts — Newsreader + Manrope
- Deployed on Netlify

---

## How It Was Built

Designed in Stitch, then manually assembled into a multi-page HTML site. No build tools, no npm — just files and a Netlify drop.
